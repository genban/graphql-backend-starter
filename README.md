### Server Starter
目前用到的技术选型，如下
> 数据交互格式： Graphql

> 框架：akka-http sangrid

> 语言：scala2.12

> 数据库： postgrel

> 库： slick, json4s, flyway

> 开发环境: docker

> 部署工具: python3 Fabric3

### 开始
```sh
#init docker
cd docker
docker-compose up

#init database
cd ../app
sbt flywayMigrate

```

### 生产环境日志
logback.pro.xml

命令： `sbt -Dlogback.configurationFile=logback.pro.xml run `

### 生产环境部署脚本
```sh
#
cd ../app
fab -H $SERVER_HOST
```
